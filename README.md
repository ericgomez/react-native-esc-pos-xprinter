# react-native-esc-pos-xprinter

This is an android xprinter driver.

## Installation

```sh
npm install react-native-esc-pos-xprinter
```

## Usage

```js
import { multiply } from "react-native-esc-pos-xprinter";

// ...

const result = await multiply(3, 7);
```
